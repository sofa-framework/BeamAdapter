Let's continue adding tubes. But instead of copying and pasting, we use the functionnalities of php with the scene file. It allows to script the scene file and repeat some parts of the scene for instance.

We created the php scene file with the extension .pscn, and then we generated the scene (extension .scn). Sofa can then work with it. In the current exemple, we generated 10 concentric tubes.

To generate scene.scn from scene.pscn, run php scene.pscn > scene.scn.
