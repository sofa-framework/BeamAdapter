InterpolateTransformUsingSpline This function provide an interpolation of a frame that is placed between node0 and node1 the location of the frame is given by baryCoord

find the spline points

The beam is very compressed => it leads to a non correct interpolation using spline (the spline formulation is based on the rest length of the beam) For a correct result, we use linear interpolation instead... (for the quaternion, we use a "simple" slerp

The position of the frame is computed using the interpolation of the spline

the tangent is computed by derivating the spline

try to interpolate the "orientation" (especially the torsion) the best possible way... (but other ways should exit...)

  1. The frame at each node of the beam are rotated to align x along n_x
  1. We use the "slerp" interpolation to find a solution "in between" these 2 solution R0, R1
  1. The result is not necessarily alligned with n_x, so we re-aligned Rslerp to obtain a quatResult.
