Detailed Description

template<class DataTypes>
class beamadapter::WireRestShape< DataTypes >

Describe the shape functions on multiple segments.

Describe the full shape of a Wire with a given set of

See also
BaseRodSectionMaterial. The wire is discretized by a set of beams (given by the keyPoints and the relatives Beam density)
d_keyPoints and @d_density are computed by method
initLengths using the set of rod sections description. This component compute the beam discretization and the shape functions on multiple segments using curvilinear abscissa.
