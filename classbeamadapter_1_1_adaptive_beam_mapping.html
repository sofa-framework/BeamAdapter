Detailed Description

template<class TIn, class TOut>
class beamadapter::AdaptiveBeamMapping< TIn, TOut >

Set the positions and velocities of points attached to a beam using linear interpolation between DOFs.

This is a component: https://www.sofa-framework.org/community/doc/programming-with-sofa/create-your-component/
