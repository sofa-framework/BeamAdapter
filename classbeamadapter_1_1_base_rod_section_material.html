Detailed Description

template<class DataTypes>
class beamadapter::BaseRodSectionMaterial< DataTypes >

Base class describing a Rod section which will define a set of beam elements.

This class provide an api to define a rod/wire section using physical and geometry parameters. The section will then be modellized by a set of beam elements. Inheriting class should provide the geometry structure:

See also
RodMeshSection to define a rod using a mesh file,
RodSpireSection or
RodStraightSection to define procedural shapes. Method
initSection and
getRestTransformOnX should be overriden to provide the correct creation and interpolation.

The rod section is described by:

  • Topology parameters: vertices and edges
    See also
    d_nbEdgesVisu,
    d_nbEdgesCollis and
    d_nbBeams
  • Geometry parameters: radius
    See also
    d_radius,
    d_innerRadius and length
    d_length
  • Mechanical parameters:
    See also
    d_poissonRatio and
    d_youngModulus
